Project 2

Rocket Booster Game

This is a Rocket Booster Game created by me. It contains 4 Levels at present, more Levels to be added in the future. In this game, you need to launch the rocket from the launch pad and fly it safely to the landing pad without touching the various obstacles that will come in the rockets trajectory. And land the rocket safely and smoothly to proceed to the next level. Don't forget to take the fuel (white circle). Controls are the space bar for power/thrust for the rocket, "A" and "D" keys for rotating the rocket in left and right directions respectively. There is a cheat code key, if you can find it.

Project 3

Flying Car

There is a 3D open World base theme of city. You can explore the world using third person character like a flying car which is a roadable aircraft. There are people, cars, trucks, etc. The features of the roadable aircraft (Flying Car) are that it can capture photos of the places/objects in view and pick/drop objects by dropping a rope with a magnet attached to its bottom. In this game, you can even call another aircraft to your current location in the world and make it follow your aircraft. The people are not static, they constantly do some movements using animations. Controls for the roadable aircraft (Flying Car):- -Up arrow key to move forward. -Down arrow key to move Backward. -Left arrow key to turn left. -Right arrow key to turn right. -Space key to fly up. -C key to capture places/objects in view. -E key to pick up the object. -R key to drop down the object. -I key to call another aircraft and make it follow your aircraft. -O key to stop that aircraft from following you.

Obstacle Game

It was my first game that I have created. It is an obstacle game. You need to go to the other end without getting in contact with the obstacles. The controls are the arrow keys.
